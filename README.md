# AstrBot 洗头频率追踪插件 (astrbot_plugin_shampoo_tracker)

## 插件简介
**AstrBot 洗头频率追踪插件**是一款专为生活记录设计的实用工具。它能够精准记录用户每次发送洗头指令的时间戳，并利用本地数据库进行持久化存储。用户可以通过简单的指令随时查看自己的洗头历史和频率统计，帮助管理个人卫生习惯。

---

## 功能说明
* **实时记录**：精确到秒级记录用户执行“洗头”操作的时间。
* **数据持久化**：使用 `aiosqlite` 异步数据库存储，确保插件重启或系统更新后数据不丢失。
* **历史追溯**：支持完整提取所有历史洗头记录。
* **周期统计**：提供直观的日期与时间展示，方便用户观察洗头规律。

---

## 插件流程
该插件遵循 AstrBot 的标准事件处理流程，内部逻辑如下：

1.  **初始化阶段**：
    * 插件启动时，在 `data/astrbot_plugin_shampoo_tracker/` 目录下初始化 SQLite 数据库文件。
    * 自动创建数据表（如果不存在），包含字段：`user_id` (用户唯一识别码), `timestamp` (洗头时间戳), `formatted_time` (格式化日期时间)。
2.  **指令捕获**：
    * 通过 `@filter.command` 监听用户输入的指令。
3.  **逻辑处理**：
    * **洗头记录**：获取当前系统时间，通过 `aiosqlite` 异步写入数据库，并绑定发送者的 `unified_msg_origin`。
    * **查询记录**：从数据库中筛选当前用户的全部记录，按时间倒序排列。
4.  **结果反馈**：
    * 将处理后的数据通过消息链（MessageChain）格式化为易读的文本，由 `yield` 返回给用户。
